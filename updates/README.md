# Updates

This folder holds versioned continuity snapshots.

## Convention

- Create one folder per improved version.
- Use the version as the folder name, for example `v7.0.0`, `v7.0.1`, `v7.1.0`.
- Put a short README in each folder with:
  - what changed
  - what is stable
  - what still needs work
  - what future AI should inspect first

## Current Folders

- `v7.0.0`
- `v7.0.1`
- `v7.0.2`

## Rule

Do not overwrite old version folders. Add a new folder for each meaningful improvement so another AI can reconstruct the project history quickly.
